


_**EL/NCL = PD × LGD × EAD = PD × (1 − RR) × EAD**_   

Where :    
**EL = Expected Loss   
NCL = Net Credit Loss   
PD = Probability of Default   
LGD = Loss Given Default    
RR = Recovery Rate (RR = 1 − LGD)   
EAD = Exposure at Default**
